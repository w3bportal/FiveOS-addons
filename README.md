# FiveOS Addons

Optional addons for [FiveOS](https://github.com/w3bportal/FiveOS), the all-in-one FiveM modding tool.

Addons add extra tools and workspaces to FiveOS. Once installed and enabled, an addon shows up under the **Addons** section of the app sidebar.

## Installing

1. Grab an addon from its release (or folder) in this repo.
2. In FiveOS, open **Settings** and enable the addon.
3. It appears in the left sidebar.
